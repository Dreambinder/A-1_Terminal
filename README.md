# A-1 Terminal

A small browser-based faux terminal for a D&D setting.  
Built with Docker, Node.js, Express, and vanilla HTML/CSS/JavaScript.

This project is an early prototype for an in-universe archive interface where players can navigate dossier trees and open files through terminal-style commands.

Current features:
- fake CRT-style interface
- simple command system
- directory navigation
- file opening from a structured data tree

This project is currently a learning exercise and prototype.
